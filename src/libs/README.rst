Libraries Directory
===================

Core reusable libraries for the entire project.

Libraries
---------
- **foundation**: Core utilities, logging, threading
- **network**: High-performance networking (libuv-based)
- **quant_core**: Quantitative finance and mathematical models
- **media_core**: Real-time communication and media processing
- **ui_kit**: Qt6-based UI component library (desktop only)

All libraries are designed to be platform-independent and thoroughly tested.
