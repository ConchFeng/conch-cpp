Android Native App
==================

Native Android application using C++ core with JNI for Android layer.

Structure
---------
- **jni/**: JNI wrappers for Java/Kotlin integration
- **src/**: C++ business logic
- **include/**: Public C++ API
- **AndroidProject/**: (To be added) Android Studio project with Kotlin UI

Building
--------
Use CMake with Android NDK or integrate with Gradle build system.
